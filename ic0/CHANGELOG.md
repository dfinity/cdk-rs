## [1.0.0] - 2025-06-25

### Changed

- Introduced new safe API in the crate root. The raw C bindings have been moved to `ic0::sys`.

### Migration guide

- Replace `ic0::*` with `ic0::sys::*`, e.g., `ic0::msg_arg_data_size()` -> `ic0::sys::msg_arg_data_size()`.
